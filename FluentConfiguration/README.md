# FluentConfigurationDemo

A clean and structured ASP.NET Core API project using Entity Framework Core with Fluent Configuration.  
This project demonstrates how to build maintainable, scalable, and readable architecture with proper configurations, services, and repositories.

---

## 🚀 Technologies Used

- ASP.NET Core Web API
- Entity Framework Core
- Fluent Configuration (Fluent API)
- SQL Server
- Repository Pattern
- Dependency Injection (DI)
