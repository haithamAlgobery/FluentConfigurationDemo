# FluentConfigurationDemo

A **clean and structured ASP.NET Core Web API project** using **Entity Framework Core** with **Fluent Configuration**.  
This project demonstrates how to build **maintainable, scalable, and readable architecture** for a real estate management system with multiple unit types (House, Apartment, Office).
